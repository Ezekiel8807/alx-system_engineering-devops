su betty: Change  user to betty
id -un: show user
groups: show all groups user is part of
chown betty hello : change hello file permission to betty
touch hello: create a hello file
chmod u+x hello : change hello to executable file
chmod u+x,g+x,o+r hello: changes user permission
chmod ugo+x hello : to give all permission to file
chmod 007 hello: all permission to others
